  Telegram 筛号工具 - 使用说明
═══════════════════════════════════════════════════════

📦 首次使用步骤：

1. 获取 Telegram API 凭证
   - 访问 https://my.telegram.org
   - 登录你的 Telegram 账号
   - 点击 "API development tools"
   - 创建应用，获取 api_id 和 api_hash

2. 配置账号信息
   - 复制 config.example.json 为 config.json
   - 编辑 config.json，填入：
     * api_id: 你的 API ID（数字）
     * api_hash: 你的 API Hash（字符串）

   - 如果需要代理，填写 proxy 部分（可选）

3. 登录 Telegram 账号
   - Windows: 双击 TelegramLogin.exe
   - macOS: 双击 TelegramLogin.app

   - 填写信息：
     * API ID 和 API Hash（会自动从 config.json 读取）
     * 手机号（带国家码，如 +8613800138000）
     * Session 名称（如 session_1）
     * 代理（可选）

   - 点击"开始登录"
   - 输入 Telegram 发送的验证码
   - 如果有两步验证，输入密码
   - 登录成功后会生成 session_*.session 文件

4. 启动主程序
   - Windows: 双击 TelegramFilter.exe
   - macOS: 双击 TelegramFilter.app

5. 开始筛选
   - 在"筛选"标签页输入或导入手机号
   - 选择目标国家（美国/中国）
   - 点击"开始筛选"

═══════════════════════════════════════════════════════

📁 文件说明：

TelegramLogin.exe / TelegramLogin.app - 登录工具（首次使用）
TelegramFilter.exe / TelegramFilter.app - 主程序
config.json - 账号配置文件（必需）
session_*.session - 登录会话文件（登录工具自动生成）
registered_*.txt - 已注册号码导出文件
unregistered_*.txt - 未注册号码导出文件
filter_progress.json - 筛选进度文件

═══════════════════════════════════════════════════════

⚠️ 注意事项：

1. API 凭证安全
   - 不要分享你的 api_id 和 api_hash
   - 不要上传 config.json 到公开位置

2. Session 文件
   - session 文件包含登录信息，请妥善保管
   - 如果更换电脑，可以复制 session 文件避免重新登录

3. 速率限制
   - Telegram 有速率限制，建议使用多个账号
   - 如果触发限制，程序会自动切换账号

4. 代理使用
   - 如果网络受限，建议配置 SOCKS5 代理
   - 代理配置错误会导致连接失败

5. 探针验证
   - 建议开启探针验证，防止批量误判
   - 探针号码应该是已知注册的号码

❓ 常见问题：

Q: 提示"手机号无效"？
A: 检查手机号格式，必须包含国家码（如 +86）

Q: 提示"连接失败"？
A: 检查网络连接，或配置代理

Q: 提示"代理认证失败"？
A: 检查代理用户名和密码是否正确

Q: 如何添加多个账号？
A: 在 config.json 的 accounts 数组中添加多个账号配置

Q: 如何更换电脑使用？
A: 复制以下文件到新电脑：
   - TelegramFilter.exe / TelegramFilter.app
   - config.json
   - session_*.session（可选，避免重新登录）

📧 技术支持：

如有问题，请检查：
1. config.json 格式是否正确（JSON 格式）
2. API 凭证是否有效
3. 网络连接是否正常
4. 代理配置是否正确

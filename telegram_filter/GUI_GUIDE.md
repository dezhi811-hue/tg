# GUI使用指南

## 🖥️ 启动GUI

```bash
# 方法1：直接运行
python gui.py

# 方法2：使用启动脚本
./run_gui.sh

# 方法3：Python脚本
python run_gui.py
```

## 📱 界面说明

GUI包含3个标签页：

### 1. 📱 筛选页面

**左侧 - 输入区域：**
- 手机号输入框：每行输入一个号码
- 📂 从文件导入：导入txt文件中的号码
- 🗑️ 清空：清空输入框
- 🌍 目标国家：选择美国🇺🇸或中国🇨🇳
- 🚀 开始筛选：启动筛选任务
- ⏹️ 停止：停止当前任务

**右侧 - 结果区域：**
- 📊 筛选进度：进度条显示完成百分比
- 📈 统计信息：实时显示已注册、在线等数据
- 📝 运行日志：详细的运行日志
- 💾 导出结果：保存为CSV或JSON

### 2. 👤 账号管理页面

**左侧 - 账号列表：**
- 显示所有已配置的账号
- ➕ 添加：添加新账号
- ✏️ 编辑：编辑选中的账号
- 🗑️ 删除：删除选中的账号

**右侧 - 账号详情：**
- 账号名称：自定义名称（如 account1）
- API ID：从 my.telegram.org 获取
- API Hash：从 my.telegram.org 获取
- 手机号：Telegram账号的手机号
- 💾 保存账号：保存当前编辑的账号

**💡 获取API凭证：**
1. 访问 https://my.telegram.org
2. 登录你的Telegram账号
3. 点击 "API development tools"
4. 创建应用，获取 api_id 和 api_hash

### 3. ⚙️ 设置页面

**速率控制设置：**
- 单账号连续请求上限：建议20-50次
- 最小延迟：建议3-5秒
- 最大延迟：建议8-15秒

**防封建议：**
- 至少配置3个账号，推荐5个以上
- 延迟时间不要设置太短
- 避免24小时连续运行
- 分批次执行，每批次后休息几小时

## 🚀 快速开始

### 第一步：添加账号

1. 切换到 "👤 账号管理" 标签页
2. 点击 "➕ 添加" 按钮
3. 填写账号信息：
   - 账号名称：account1
   - API ID：12345678
   - API Hash：abcdef1234567890...
   - 手机号：+12025551234
4. 点击 "💾 保存账号"
5. 重复以上步骤添加更多账号（建议3-5个）

### 第二步：输入号码

1. 切换到 "📱 筛选" 标签页
2. 在左侧输入框中输入号码（每行一个）：
   ```
   +12025551234
   2025551235
   (202) 555-1236
   ```
3. 或点击 "📂 从文件导入" 导入txt文件

### 第三步：开始筛选

1. 选择目标国家（美国🇺🇸 或 中国🇨🇳）
2. 点击 "🚀 开始筛选"
3. 首次运行会要求输入每个账号的验证码
4. 查看右侧的实时进度和日志

### 第四步：导出结果

1. 筛选完成后，点击 "💾 导出结果"
2. 选择保存位置和格式（CSV或JSON）
3. 结果文件包含所有筛选信息

## 📊 结果说明

CSV文件包含以下字段：
- **phone**: 格式化后的号码
- **original_phone**: 原始输入号码
- **country**: 国家代码（US/CN）
- **registered**: 是否注册（True/False）
- **user_id**: Telegram用户ID
- **username**: 用户名
- **first_name**: 名字
- **last_name**: 姓氏
- **status**: 在线状态
  - 🟢 online - 当前在线
  - 🟡 recently - 最近在线
  - 🟠 within_week - 一周内在线
  - 🔴 within_month - 一个月内在线
  - ⚫ offline - 离线
  - ⚪ long_ago - 很久未上线
- **last_seen**: 最后上线时间
- **is_bot**: 是否为机器人
- **error**: 错误信息

## ⚠️ 常见问题

**Q: 点击"开始筛选"没反应？**
A: 请先在"账号管理"中添加至少一个账号。

**Q: 提示"未找到tkinter"？**
A: 
- macOS: `brew install python-tk`
- Ubuntu: `sudo apt-get install python3-tk`
- Windows: tkinter通常已包含在Python中

**Q: 首次运行要求输入验证码？**
A: 这是正常的，Telegram会发送验证码到你的手机，输入后会保存session，下次不需要再输入。

**Q: 如何提高筛选速度？**
A: 添加更多账号（3-5个），系统会自动轮换使用。

**Q: 账号被封了怎么办？**
A: 降低请求频率，增加延迟时间，添加更多账号轮换。

## 💡 使用技巧

1. **批量导入**：准备一个txt文件，每行一个号码，使用"从文件导入"功能
2. **多账号**：至少配置3个账号，可以大幅提高效率和安全性
3. **分批执行**：不要一次筛选太多号码，建议每批100-500个
4. **定期休息**：每筛选一批后，休息几小时再继续
5. **保存配置**：账号信息会自动保存到config.json，下次启动自动加载

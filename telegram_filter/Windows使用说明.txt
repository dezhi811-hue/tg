Telegram 筛号工具 - Windows 使用说明
========================================

## 首次使用步骤

### 1. 准备工作
- 解压 TelegramFilter-Windows.zip 到任意文件夹
- 确保文件夹中有：
  * TelegramFilter.exe (主程序)
  * config.json (配置文件)

### 2. 配置账号信息
编辑 config.json 文件，填入你的 Telegram 账号信息：

```json
{
  "accounts": [
    {
      "name": "1",
      "api_id": "你的API_ID",
      "api_hash": "你的API_HASH",
      "phone": "+86手机号",
      "proxy": {
        "host": "",
        "port": 1080,
        "username": "",
        "password": ""
      }
    }
  ]
}
```

### 3. 获取 API 凭证
访问 https://my.telegram.org/apps
- 登录你的 Telegram 账号
- 创建应用获取 api_id 和 api_hash

### 4. 首次登录
**重要：首次使用必须先登录账号！**

方法一：使用命令行登录
1. 打开命令提示符 (cmd)
2. 进入程序目录：cd 你的文件夹路径
3. 运行：python login.py
4. 输入验证码和两步验证密码（如果有）

方法二：使用 Python 脚本
如果没有安装 Python，需要先安装：
- 下载 Python 3.8+ from https://www.python.org/
- 安装依赖：pip install telethon pysocks

### 5. 运行程序
双击 TelegramFilter.exe 启动

## 常见问题

### ❌ 无法连接 Telegram 服务器

**原因 1：没有 session 文件**
- 解决：必须先运行 login.py 登录账号
- session 文件会自动生成在程序目录

**原因 2：网络问题**
- 中国大陆需要代理才能连接 Telegram
- 在 config.json 中配置代理：
  ```json
  "proxy": {
    "host": "127.0.0.1",
    "port": 1080,
    "username": "",
    "password": ""
  }
  ```

**原因 3：防火墙阻止**
- 检查 Windows 防火墙设置
- 允许 TelegramFilter.exe 访问网络

**原因 4：Session 文件过期**
- 删除 session_*.session 文件
- 重新运行 login.py 登录

### ❌ 密码错误提示

**原因：账号设置了两步验证但 session 失效**
- 解决：重新登录账号
- 删除对应的 session 文件
- 运行 login.py 重新登录

### ❌ 缺少依赖模块

如果提示缺少模块，安装：
```
pip install telethon pysocks PyQt5
```

## 代理设置说明

### 不使用代理（直连）
```json
"proxy": {
  "host": "",
  "port": 1080,
  "username": "",
  "password": ""
}
```

### 使用 SOCKS5 代理
```json
"proxy": {
  "host": "127.0.0.1",
  "port": 1080,
  "username": "",
  "password": ""
}
```

### 使用需要认证的代理
```json
"proxy": {
  "host": "代理地址",
  "port": 端口,
  "username": "用户名",
  "password": "密码"
}
```

## 文件说明

- TelegramFilter.exe - 主程序
- config.json - 配置文件（必需）
- session_*.session - 登录会话文件（自动生成）
- *.txt - 筛选结果文件
